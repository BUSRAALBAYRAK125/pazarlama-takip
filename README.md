# pazarlama-takip
Python ile geliştirilmiş müşteri yönetim uygulaması
Bu proje, CustomTkinter ile geliştirilmiş kullanıcı arayüzü ve MySQL veritabanı ile çalışan bir müşteri, ödeme ve pazarlama yönetimi uygulamasıdır.

Özellikler:

Müşteri ekleme, bilgilerini güncelleme ve listeleme

Ödeme ekleme, güncelleme ve dekont yönetimi

Pazarlamacı ve iş takibi

Dosya işlemleri ve dekont kaydı

Veritabanı işlemleri için hazır sınıf yapısı (vtclass.py)

Kullanılan teknolojiler:

Python 3.x

CustomTkinter (GUI)

MySQL (veritabanı)

shutil ve os (dosya işlemleri)

Amaç:
Küçük işletmeler veya eğitim amaçlı kullanım için, müşteri ve ödeme verilerini kolayca yönetebilecek bir masaüstü uygulaması sunmak.
